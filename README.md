# Mail IQ

**Тестовое задание для web-мастера в женскую онлайн-школу.**

Адаптивная страница, с рабочими элементами, реализованными на jQuery.

1. Валидная верска
2. Написана по БЭМ-методологии
3. Адаптивная верска для разрешений от 320px
4. Минимизирован вес изображений за счет использования формата WEBP
5. Динамические элементы(кнопки, дропдауны, аккордеон) реализован на jQuery
6. Для активных элементов(кнопки, ссылки) реализованы состояния hover, active с плавным переходом

> Примерное время выполнения задачи - 35-40 часов. Значительно быстрее выполнил бы при наличии макета в Figma. 

> Не использвал Bootstrap, так как почти не работал с ним, но имею представление и, если необходимо, могу переделать часть стилей через него. 

> Некоторые стили, а так же отступы, цвета, размеры и пр. брал с оригинального [сайта](https://sobakarisovaka.ru/pages/petersburgtime/).

> Использвал нейросеть для написания скриптов jQuery и решению некторых вопросов по оптимальным решениям для стилизации.
